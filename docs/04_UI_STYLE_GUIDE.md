# UI 风格指南

## 总体气质

1. 采用现代简洁风格；
2. 保持轻量桌面工具感；
3. 不使用传统老旧 Windows 表格风；
4. 使用卡片式布局；
5. 使用圆角；
6. 保持低干扰；
7. 浮窗要小、轻、可折叠；
8. 主窗口采用左侧分类、中间列表、右侧详情结构；
9. 支持浅色和深色主题；
10. 不做花哨动画。

## 交互原则

- 优先快速记录和清晰阅读，控制视觉噪声。
- 浮窗只承载高频、必要操作，避免堆叠完整主窗口能力。
- 主窗口的信息层级清楚，卡片、间距和圆角保持一致。
- 动效仅用于必要的状态反馈，不作为视觉展示手段。

## 内容边界

UI 不得出现或预留 AI 自动整理、AI 每日复盘、行情、荐股、券商同步、自动交易、独立图片库、人工总结模块的入口、卡片、按钮或占位页面。
