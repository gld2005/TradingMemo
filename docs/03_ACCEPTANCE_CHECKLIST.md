# Batch 验收清单

每个 Batch 只有在对应标准全部满足、未突破全局范围禁区并经用户确认后，才视为通过。

## Batch 0：项目准备与规范冻结

- [ ] `README.md` 存在，并准确描述项目定位；
- [ ] `SKILL.md` 存在，并明确约束开发 Agent 不扩大需求；
- [ ] `docs/00_PROJECT_OVERVIEW.md` 存在；
- [ ] `docs/01_PRODUCT_SCOPE.md` 存在；
- [ ] `docs/02_BATCH_PLAN.md` 存在，并只包含确认后的 8 个功能 Batch；
- [ ] `docs/03_ACCEPTANCE_CHECKLIST.md` 存在；
- [ ] `docs/04_UI_STYLE_GUIDE.md` 存在；
- [ ] `docs/05_TECHNICAL_BOUNDARY.md` 存在；
- [ ] 所有项目文档均明确禁止 AI、行情、荐股、券商同步、独立图片库和人工总结模块，并禁止自动交易；
- [ ] 没有实现任何业务功能；
- [ ] 没有提前进入 Batch 1。

## Batch 1：项目骨架 + 现代简洁 UI 基础

- [ ] Windows 桌面应用骨架可启动和正常退出；
- [ ] 统一的基础视觉规范可被页面复用；
- [ ] 浅色和深色主题基础可验证；
- [ ] 未实现 Batch 2 及之后的业务功能。

## Batch 2：桌面浮窗核心功能

- [ ] 浮窗可独立显示、隐藏、拖动，并在关闭时转为隐藏；
- [ ] 浮窗支持置顶、展开和迷你模式，尺寸和交互符合小、轻、低干扰要求；
- [ ] 主窗口入口和 `Alt + J` 可切换浮窗，快捷键注册失败时有降级提示；
- [ ] 快速文字输入、静态分类和保存反馈仅保留在前端临时状态；
- [ ] 浮窗的基础 Windows 桌面行为稳定；
- [ ] 未实现本 Batch 之外的数据和笔记功能。

## Batch 3：本地数据存储

- [x] 本地数据可保存、读取和更新；
- [x] 数据存储不依赖网络、云端或登录；
- [x] 本地数据位置和基本异常行为可验证；
- [x] 未接入任何外部 API。

## Batch 4：图文记录功能

- [x] 可创建纯文字、图文和纯图片笔记，不包含编辑功能；
- [x] 可在浮窗粘贴和拖入图片，并在保存前预览或移除草稿；
- [x] 图片作为笔记附件保存到本地，应用重启后可在今日记录重新查看；
- [x] 支持格式、大小、数量、保存失败和本地文件丢失等基础边界提示；
- [x] 未建立独立图片库，未实现 OCR、AI 识别或后续 Batch 功能。

## Batch 5：分类与标签系统

- [x] 可创建、修改和使用分类；
- [x] 可创建、修改和使用标签；
- [x] 笔记可维护股票名称/代码字段；
- [x] 所有组织操作均由用户手动完成。

## Batch 6：主窗口知识库页面

- [x] 主窗口具有左侧分类、中间列表、右侧详情结构；
- [x] 可浏览和维护已有本地笔记；
- [x] 界面符合 UI 风格文档；
- [x] 页面不展示行情、荐股或交易内容。

## Batch 7：搜索与筛选

- [x] 可按标题、正文、股票、分类名称和标签名称进行本地轻量搜索，支持中文并忽略英文大小写；
- [x] 可按分类、多标签 AND、股票名称/代码、创建日期和是否带图片筛选；
- [x] 搜索与筛选按 AND 组合，结果数量、选中详情、空状态和数据刷新行为正确；
- [x] 左侧分类与筛选状态一致，支持单项清除和一键清空条件；
- [x] 自定义日期范围包含起止日期，反向范围有友好提示且不导致页面崩溃；
- [x] 不进行外部搜索、全文索引、智能推荐、AI 或远程数据处理。

## Batch 8：设置、导出、打包

- [ ] 设置项可保存并在重启后生效；
- [ ] 导出、备份和恢复流程可验证；
- [ ] Windows 安装或可分发包可在目标环境运行；
- [ ] 第一版范围内功能通过回归检查。

## 全局否决项

任一 Batch 一旦加入 AI 自动整理或每日复盘、行情、荐股、券商同步、自动交易、独立图片库、人工总结模块，即视为不通过。

## Batch 8 详细验收补充

- [ ] 设置页包含外观、浮窗、默认记录、数据位置、导出与备份、关于项目六个区域；
- [ ] 主题即时生效并持久化；快捷键修改失败可回退且可恢复 `Alt+J`；
- [ ] 默认分类来自本地分类，失效时回退未分类；数据目录可查看和打开；
- [ ] Markdown/JSON 导出结构完整，附件复制且 Markdown 使用相对路径；
- [ ] 备份包含 `notes.json`、`settings.json`、`attachments/`；
- [ ] 恢复前确认和安全备份，无效格式拒绝，失败回滚；
- [ ] 首次引导可关闭并持久化，关于区域含本地保存说明和免责声明；
- [ ] Windows x64 离线安装包可生成，无自动更新、遥测和远程发布。
