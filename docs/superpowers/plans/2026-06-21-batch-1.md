# Batch 1 Project Skeleton Implementation Plan

> **For agentic workers:** Execute this plan inline in the current workspace. Subagents are intentionally not used because the user prohibited unnecessary multi-agent work.

**Goal:** Build a runnable Electron Windows desktop skeleton with reusable modern UI foundations, four placeholder pages, a static floating-window placeholder, and temporary light/dark theme state.

**Architecture:** Electron owns two renderer windows backed by one Vite/React entry. A query parameter selects the main shell or floating placeholder. React local state handles navigation and theme without persistence, while focused components and shared CSS tokens provide reusable visual structure.

**Tech Stack:** Electron, React, TypeScript, Vite, Vitest, Testing Library, CSS
