# 项目定位

## 项目定义

本项目是 **Windows 桌面浮窗版 A 股学习笔记软件**。

核心场景是：用户在看盘、学习或阅读资料时，使用小巧、低干扰的桌面浮窗快速记录文字和图片；之后再通过主窗口整理和查找这些学习记录。

## 第一版目标

第一版只提供本地记录、分类、搜索、筛选、导出和备份能力。产品优先解决“随手记下”和“日后找回”两个问题，不扩展到投资决策或交易执行。

## 产品身份边界

- 本项目不是行情软件，不提供实时或延迟行情。
- 本项目不是荐股软件，不生成或展示买卖建议。
- 本项目不是交易软件，不连接交易账户，不执行交易。
- 本项目不是 AI 自动复盘工具，不做 AI 自动整理或 AI 每日复盘。

## 明确禁止

第一版明确禁止并不包含：AI 自动整理或每日复盘、行情接入、荐股、券商同步、自动交易、独立图片库、人工总结模块。图片只作为笔记内容的附件保存和查看。
