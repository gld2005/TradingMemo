# src

本目录包含 React 应用骨架、基础组件、主窗口页面、主题样式，以及分类标签管理、浮窗图文草稿、今日记录和 Batch 7 知识库搜索筛选能力。

当前代码提供主窗口、分类与标签管理、知识库三栏浏览、笔记详情/编辑/删除，单个桌面浮窗的置顶、显示/隐藏、拖动和折叠/展开交互，以及图文保存、分类标签选择、股票名称/代码记录和今日记录展示。知识库基于已加载的本地 Note、Category 和 Tag 数组完成文本、分类、多标签 AND、股票、创建日期和带图条件的组合过滤。真实笔记、分类、标签与附件通过 Electron 安全桥接保存到本地用户数据目录；本目录不直接执行文件系统操作。

当前仍不包含导出、备份、恢复、完整设置、打包发布、独立图片库、OCR、AI 图片识别或其他 Batch 8 功能。
